This is a sidekick-type app for Project: Gorgon, that you would run along with the game that will give a few added QoL features to chat, combat xp logging, and loot visibility. The tool contains many options to choose sounds to play when certain alerts occur, volume control, and many other customizable options.

**This requires chat logs to be enabled, which requires VIP, to my knowledge.**

1. Download the current release, the exe file. You can place this file anywhere.
2. Optional custom sounds: create a folder named custom_sounds in the same directory as the script or .exe, then add .wav, .mp3, or .ogg files.
3. Sounds from custom_sounds are added to the sound dropdowns in the app. If a filename exists in both sounds and custom_sounds, the bundled sounds version is used.
4. Enable Chat Logging (may require VIP)
5. Open the game, then run the .exe. Once logged in, you can configure your anchors.
6. When you are ready, click Start Monitoring. This will start to read from the chat log to queue alerts and other features.
